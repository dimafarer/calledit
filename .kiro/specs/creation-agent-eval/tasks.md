# Implementation Plan: Creation Agent Eval

## Overview

Build the v4 creation agent eval framework: AgentCore backend → 6 Tier 1 deterministic evaluators → 2 Tier 2 LLM judge evaluators → CLI runner → JSON report generation. All code lives under `eval/` and invokes the deployed agent via HTTP streaming (no local agent, no mocks). Depends on V4-7a-1 (golden dataset reshape) for `smoke_test` flags in the dataset.
